

《数据结构实践教程》
配 套 光 盘
     
　　本光盘是清华大学出版社出版的《数据结构实践教程》（ISBN：978-7-7-302-21476-2）的配套光盘。
　　
　　本练习光盘收录了《数据结构实践教程》的所有源程序以及源程序目录和使用方法。
     
　　本光盘中共有11个文件夹和4个文件。
1.  readme.doc和readme.txt文件。两个文件的内容完全相同，是光盘的简单介绍。
2. 源程序目录.doc和源程序使用说明.doc两个文件。
3. 与教材各章对应的11个文件夹，存放源程序。
